This is an expansion pack for the Time Fantasy RPG assets.
The sprites in this pack will fit with all of my graphics in this style.

This pack is a collection of sprites that shows three new fantasy races based on animals. The three tribes are:
	- Rat Men
	- Insects
	- Frogs

Each tribe has a set with 8 regular character sprites, as well as a set of base-sprites with some colors variants. 

In addition, each tribe has one special champion character. The champions also have additional animations
for emotions, and a sheet with side-view RPG-style battle animations.

The included side-view battle sprites are arranged in the format used by RPG Maker MV.
