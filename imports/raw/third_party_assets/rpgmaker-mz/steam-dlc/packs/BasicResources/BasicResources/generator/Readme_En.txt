This folder contains additional Character Generator assets for RPG Maker MZ.
You can use them with the Character Generator by following the steps below.

1. Terminate RPG Maker MZ.
2. Open the RPG Maker MZ installation folder (for the Steam version, you can open it by right-clicking on "RPG Maker MZ" in your Steam library, selecting "Manage", and then selecting "Browse Local Files".)
3. Make a backup of the "generator" folder in the RPG Maker MZ installation folder.
4. Open the generator/Face/Female folder.
5. Copy all the files in the dlc/BasicResources/generator/Face/Female folder and paste them to the folder opened in step 4 (if files with the same name already exist, rename either of them).
6. Repeat the same procedure for the generator/Face/Kid and generator/Face/Male folders.
7. Repeat the same procedure for the generator/SV, generator/TV, generator/TVD, and generator/Variation folders.
8. Run RPG Maker MZ and open "Character Generator". Now, you can choose the new assets.

The dlc/BasicResources/generator/odd-eye_bodies folder contains body images for so-called "odd-eye" characters with the left and right eye colors different. By using these assets, you can make Walking Characters and Battlers with odd-eyes, too.
Before using them, please back up the following files.
・"SV_body_p01.png" and "SV_body_p01_c.png" that already exist in the "Female", "Kid" and "Male" folders in the generator/SV folder
・"TV_body_p01.png" and "TV_body_p01_c.png" that already exist in the "Female", "Kid" and "Male" folders in the generator/TV folder

After that, replace the files with the same name in the "Female", "Kid" and "Male" folders in the dlc/BasicResources/generator/odd-eye_bodies with the files above. There are 4 color variants for the left eye.
After creating the odd-eyed characters, please return to the backed up files.
