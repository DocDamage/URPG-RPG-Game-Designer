# external-repos

Governed root for mirrored external repositories referenced by the URPG external intake program.

This directory is intentionally metadata-first. Repositories may only be mirrored here after:

- a recorded legal disposition in `docs/external-intake/license-matrix.md`
- a recorded handling path in `docs/external-intake/repo-watchlist.md`
- any required fixture or attribution metadata has been defined

See [docs/external-intake/URPG_repo_intake_plan.md](../../docs/external-intake/URPG_repo_intake_plan.md) and [docs/PROGRAM_COMPLETION_STATUS.md](../../docs/PROGRAM_COMPLETION_STATUS.md) (`P3-02`, Phase 4.1).
