# Security Policy

## Supported Versions

URPG is pre-1.0 software. Security updates are provided for the latest public
release and the current `main` branch.

| Version | Supported |
| ------- | --------- |
| Latest public release | Yes |
| `main` branch | Yes |
| Older prerelease builds | No |

## Reporting a Vulnerability

Report suspected vulnerabilities through GitHub private vulnerability reporting
for this repository when available. If private reporting is unavailable, open a
GitHub issue with a minimal description and request a private maintainer contact
before sharing exploit details.

Please include:

- Affected version or commit.
- Platform and build configuration.
- Reproduction steps or a proof of concept.
- Expected impact and whether the issue is already public.

The maintainers will acknowledge valid reports, investigate reproducible issues,
and coordinate fixes before publishing detailed advisories.
